    iTerm2 配置和美化

    3/7/2021 效率

    # 安装iTerm2

    首先，下载 (opens new window)iTerm2来替换默认终端。 什么是iTerm2？ iTerm2是默认终端的替代品，也是目前Mac系统下最好用的终端工具，集颜值和效率于一身；

    # 调整Status Bar

    可以在Profiles选项卡，Session页面最底部看到开启选项。Status bar enabled 选项，勾选上即可打开。点击右边的 Configure Status Bar 按钮可设置显示的内容。 可以看到能显示的内容非常多，把上方要显示的内容拖动到下方 Active Components 区域即添加。 在Preference页面中点击Appearance选项卡，可以设置Status bar的位置，修改 Status bar location，我这里改到Bottom底部。

    # 更改配色(可选)

    我比较喜欢 Dracula (opens new window) 配色方案，包括VS Code使用的都是这个配色方案。

    解压后更换 Preferences-Profiles-Color-Color Presets-Import

    # 设置zsh主题

    前提： 安装了zsh; 安装了powerline (opens new window)字体;

    只需要修改 .zshrc 文件中的 ZSH_THEME="xx" 属性即可。

    我目前使用的主题是：spaceship (opens new window)

    主题安装教程看这里 (opens new window);

    按照github项目说明中的方法配置即可，或者参考本文下面的主题配置

    然后把iTerm的字体设置为powerline，包括Font和Non-ASCII Font;

    • 下载主题文件：

        git clone https://github.com/denysdovhan/spaceship-prompt.git "$ZSH_CUSTOM/themes/spaceship-prompt" --depth=1

    • 创建软连接：

        ln -s "$ZSH_CUSTOM/themes/spaceship-prompt/spaceship.zsh-theme" "$ZSH_CUSTOM/themes/spaceship.zsh-theme"

    • 修改zsh配置：

        vim ~/.zshrc

      vim操作点这里 (opens new window)

    • 修改：

        ZSH_THEME="spaceship"

    • 生效设置：

       source ~/.zshrc

    # 设置Oh-My-Zsh

    什么是Oh-My-Zsh (opens new window)?

    Oh My Zsh是一款社区驱动的命令行工具，它基于zsh命令行，提供了主题配置，插件机制，大大提高了可玩（用）性。提升使用效率还能使人愉快hhh;

    输入如下命令：

       sh -c "$(curl -fsSL https://raw.githubusercontent.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"

    # 配置命令自动提示

    关于插件的配置，如果要看目前安装了哪些插件，可以在.zshrc中搜索plugins关键字

      plugins=(
    git
  )

    • 编辑 ~/.zshrc 配置文件，添加到plugins列表中;

        plugins=(zsh-autosuggestion)

    # 配置语法高亮显示

    • 推荐使用HomeBrew进行安装：

        brew install zsh-syntax-highlighting

    • 在 ～/.zshrc 尾部，插入一行：

        source /usr/local/share/zsh-syntax-highlighting/zsh-syntax-highlighting.zsh

    • 生效设置：

       source ~/.zshrc

    好了～打开iterm 享受花里骨哨吧～
